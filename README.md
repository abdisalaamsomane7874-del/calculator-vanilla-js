# JavaScript Smart Calculator

A feature-rich calculator built using vanilla JavaScript.

## Features
- Basic arithmetic operations
- Input validation
- Percentage calculation
- Last answer (ANS)
- Keyboard support
- Calculation history (last 5)
- Persistent data using localStorage

## Technologies
- HTML
- CSS
- JavaScript
